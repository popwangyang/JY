# jy-admin
#鲸易版权-平台管理后台

##项目环境
vue-cli3.0脚手架

使用模块

[打包gzip模块](https://github.com/webpack-contrib/compression-webpack-plugin）
[图片压缩模块](https://github.com/Klathmon/imagemin-webpack-plugin）
[生产环境去除console.log输出](https://github.com/webpack-contrib/terser-webpack-plugin）

## 项目设置
```
yarn install
```

